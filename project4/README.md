# Ultrasonic Trashcan

Programmed an ESP32 to control a servo motor using an ultrasonic distance sensor and a potentiometer to create an automatic trashcan.

## Hardware Components
- ESP32-C3Fx4 
- Ultrasonic sensor (HC-SR04)  
- Micro servo motor  
- Potentiometer, 10kΩ  
- Resistor, 4.7kΩ   
- Cardboard box  
- Tape
